# 贝壳｜房产平台｜后端开发｜question-bank（候选）

> 来源说明：本题库基于贝壳公开可访问的后端 / Java 面经与技术整理页保守聚合而成，优先映射 `房产平台 / 后端开发` 方向。当前版本**不冒充逐轮完整核实 JSON**，而是作为首个高质量 `question-bank` 候选，便于后续继续补正式题库或抽取单帖 JSON。
>
> 证据边界：本版本主要依赖牛客公开面经、CSDN 面经记录、小林coding 的公开整理页；其中“推荐策略”公开高质量单帖证据偏弱，因此本轮不强做推荐策略方向，优先落地后端开发题库。
