## 目录结构

```
companies/
  公司名称/
    事业群/
      部门/
        question-bank.md          # Markdown 题库
        interview-experiences/
          2024-公司-部门-岗位-面经.json  # 结构化面经
```

- 第一层只有公司名称
- 第二层是事业群 / 业务线
- 第三层是部门，下设题库与面经

## 数据设计

**题库 (`question-bank.md`)**
按专题分组，覆盖基础八股、算法题、系统设计、项目深挖、业务场景题，含追问链路。

**面经 (`interview-experiences/*.json`)**
一场面试一份 JSON，保留：面试轮次、每轮题目、题目感悟、整场面试感悟、来源信息。

JSON Schema 示例见 `docs/json-schema.example.json`。
