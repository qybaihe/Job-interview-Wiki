## 高频题
- MySQL 为什么会出现深分页慢查询？怎么优化？
- InnoDB 的 MVCC 是怎么实现可重复读的？
- Redis 持久化 RDB 和 AOF 怎么选？混合持久化适合什么场景？
- 讲一下缓存一致性问题，先更新数据库还是先删缓存？
- 分库分表后全局唯一 ID 怎么设计？
- MQ 为什么会重复消费？如何保证消费者幂等？
- RPC 调用链里超时、重试、熔断、隔离分别解决什么问题？
- Java 锁升级过程是怎样的？synchronized 和 ReentrantLock 的区别是什么？
- ThreadLocal 使用不当为什么会内存泄漏？
- Spring 事务什么时候会失效？
- 什么是热点 key？什么是大 key？如何发现和治理？
- 如何设计接口限流、防刷、降级策略？
