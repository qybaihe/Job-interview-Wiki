# Job Interview Wiki

一个面向中文求职场景的结构化面试资料仓库。

目标：
- 沉淀 **公司 / 事业群 / 部门 / 岗位方向** 维度的面试经验
- 将可复用的高频题整理为 **Markdown 题库**
- 将具体面试经历与题目映射整理为 **JSON 面试经验数据**
- 最终形成可检索、可扩展、可持续补充的面试知识库

## 设计原则

### 1. 题库与面经分离
- **题库**：用 Markdown 存储，适合人读、人改、人总结
- **面试经验**：用 JSON 存储，适合结构化整理、聚合、后续检索

### 2. 结构层级
仓库按如下层级组织：

`公司名称 / 事业群 / 部门 / 题库与面经并列`

示意：

```text
companies/
  字节跳动/
    抖音电商/
      商业化技术/
        question-bank.md
        interview-experiences/
          README.md
          sample.json
```

### 3. JSON 面经结构
每一份面经 JSON 记录：
- 面试基础信息
- 题目列表
- 每个题目对应的回答思路 / 感悟
- 最终整场面试的总感悟
