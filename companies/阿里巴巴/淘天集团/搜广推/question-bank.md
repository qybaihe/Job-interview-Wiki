136. 阿里妈妈搜索广告的经典召回方案是什么？（两段式：深度语义模型解决相关性 + 基于协同过滤的深度价值模型解决流量价值；两段式目标融合容易顾此失彼）
137. 搜索广告中的 query 改写和自然搜索的 query 改写有什么不同？（搜索广告改写需要在扩展流量覆盖的同时保证广告相关性，改写过度会导致不相关广告展示，影响用户体验和广告主 ROI）

### LLM 在搜索广告中的应用（2024-2025 重点方向）
138. LLM 在搜索广告 query 改写中的应用？（生成式改写是 LLM 在搜索广告中第一个上线项目；电商广告领域知识的 SFT 和在线动态 RAG 是迭代初期的常规优化手段；核心优势是处理长尾 query 的语义理解能力）
139. LLM 生成式改写相比传统改写方法（session 共现、word2vec、seq2seq）的优势和挑战？（优势：世界知识带来的语义理解和推理能力，长尾 query 覆盖率大幅提升；挑战：推理延迟、成本、幻觉导致不相关改写、线上 serving 架构适配）
140. 搜索广告中 LLM 的落地路径？（渐进式优化：先用 LLM 做离线数据增强/标注 → 再做在线 query 改写 → 最终探索端到端生成式广告检索；弱化归纳偏置、强化数据驱动是主线）

### CTR 预估模型的稠密化（Deeper 方向）
141. 搜索广告 CTR 预估模型的两个核心提效方向是什么？（Embedding 建模和用户行为兴趣建模；这两个方向经过多年迭代积累，形成最具迭代生命力的提效路线）
142. 什么是 CTR 模型的稠密化（Deeper）方向？和传统稀疏模型有什么区别？（稠密模型用几十层深度网络做语义理解，稀疏模型用 Embedding + 浅层交叉做统计判别；稠密模型在语义理解上更强但训练和推理成本更高）
143. 搜索广告场景下，稠密模型和稀疏模型如何协同？（稀疏模型擅长统计判别和 ID 特征记忆，稠密模型擅长语义泛化；工业界通常用稀疏模型做主模型 + 稠密模型做特征增强/Embedding 生成）

### 多模态 Embedding 在搜索广告中的应用
144. 搜索广告中多模态 Embedding 的核心挑战是什么？（承载语义信息的内容空间与承载协同信息的兴趣空间如何有效对齐）
145. 多模态 Embedding 的 Pre-train 和 Post-train 分别负责什么？（Pre-train 职责是 Encode，负责内容空间的理解与迁移，关注图文是什么，将开源世界知识往电商知识迁移；Post-train 职责是 Align，负责内容空间与兴趣空间的对齐）
146. CLIP/BLIP/BLIP2 等多模态模型在搜索广告中如何应用？（广告创意理解、广告-query 语义匹配、广告素材质量评估；需要在通用多模态能力基础上做电商领域适配）

### 搜索广告系统设计题
147. 设计一个搜索广告系统，要求同时优化广告相关性和商业价值，支持百万级广告主、亿级 query。（核心：query 理解 → 广告召回（相关性+价值双通道）→ 粗排 → 精排（eCPM）→ 竞价计费 → 混排展示）
148. 设计一个搜索广告的 query 改写系统，要求支持传统改写和 LLM 生成式改写的混合架构，保证延迟 < 50ms。（核心：离线 LLM 改写缓存 + 在线轻量模型实时改写 + 改写质量评估与过滤 + A/B 实验框架）
149. 如何设计搜索广告和自然搜索结果的联合排序系统？（核心：统一排序框架、广告加载率控制、用户体验约束、广告位竞价与自然位的 Pareto 优化）

### 面试回答建议
- 搜索广告方向面试区别于信息流广告：核心考察 query 理解能力、相关性与商业价值的平衡、搜索广告特有的召回和排序策略
