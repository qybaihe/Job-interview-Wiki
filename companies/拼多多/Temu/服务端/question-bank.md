## 1. MySQL

- 索引是什么数据结构？索引是不是越多越好？为什么？
- MySQL 可能出现什么并发相关问题？脏读、不可重复读、幻读的区别？
- 可重复读下的幻读问题举例？InnoDB 怎么解决？（MVCC + 间隙锁）
- MVCC 原理？隐藏列（trx_id、roll_pointer）？ReadView 判断可见性？
- UndoLog 和 RedoLog 各自的作用？为什么有了 UndoLog 还需要 RedoLog？
- 分表和分库的区别？什么时候分表什么时候分库？
- MySQL 执行一条 SQL 的完整流程？慢查询怎么排查？
- 联合索引的最左匹配原则？索引下推（ICP）了解吗？

- Redis 持久化机制？RDB 和 AOF 的区别？
