## 13. Go 语言专项（推荐架构后端必考，来源：2024-2025 牛客/CSDN 字节推荐架构面经）

> 字节推荐架构后端主要使用 Go 语言，以下为高频 Go 面试题。

### 13.1 GMP 调度模型
- Go 的 GMP 调度模型是什么？G、M、P 分别代表什么？
- Goroutine 和 OS 线程的区别？Goroutine 的栈大小？
- P 的数量由什么决定？GOMAXPROCS 设置为多少合适？
- Goroutine 的调度时机有哪些？（系统调用、channel 阻塞、主动让出 runtime.Gosched）
- 当一个 Goroutine 执行系统调用阻塞时，调度器怎么处理？（M 和 P 解绑，P 绑定新 M 继续执行其他 G）

### 13.2 Channel 与并发
- Channel 的底层实现？hchan 结构体包含哪些字段？
- 有缓冲和无缓冲 channel 的区别？各自适用什么场景？
- 向已关闭的 channel 发送数据会怎样？从已关闭的 channel 读取呢？
- select 多路复用怎么用？所有 case 都就绪时怎么选？都不就绪呢？
- 如何用 channel 实现一个 worker pool？推荐架构中哪些场景会用到？

### 13.3 内存管理与 GC
- Go 的垃圾回收算法？三色标记法的原理？写屏障的作用？
- Go GC 的 STW（Stop The World）发生在什么阶段？如何减少 STW 时间？
- GOGC 参数的含义？推荐架构在线服务中 GOGC 一般设多少？为什么？
- Go 的内存分配器（tcmalloc 风格）？mcache、mcentral、mheap 的关系？
- 如何排查 Go 程序的内存泄漏？pprof heap profile 怎么看？

### 13.4 Goroutine 泄漏与排查
- Goroutine 泄漏的常见原因？（channel 未关闭/未消费、无超时的阻塞调用、WaitGroup 使用不当）
- 如何监控 Goroutine 数量？runtime.NumGoroutine() 的使用？
- pprof 的 goroutine profile 怎么分析？如何定位泄漏的调用栈？

### 13.5 追问链路
- GMP 调度→P 的数量→系统调用时的 M/P 解绑→网络 IO 的 netpoller 优化
- Channel→底层 hchan→发送/接收等待队列→select 实现→runtime 层面的调度
- GC 三色标记→写屏障→STW 优化→GOGC 调参→在线服务 P99 与 GC 的关系

## 14. 特征平台深度专项（推荐架构核心，来源：2024-2025 牛客/CSDN/力扣字节推荐面经）

> 特征平台是推荐架构后端的核心基础设施，以下为深度追问题。

### 14.1 特征分层架构
- 离线特征、近线特征、实时特征分别怎么计算和存储？
- 离线特征（T+1）的计算链路？Hive/Spark → HBase/Redis 的同步怎么做？
- 近线特征（分钟级）用 Flink 窗口聚合怎么设计？窗口类型怎么选？
- 实时特征（秒级）的计算和存储？Flink 实时流 → Redis/本地缓存的延迟怎么控制？
- 三层特征在线 Serving 时如何并行拉取和融合？超时降级策略？

### 14.2 特征存储选型
- 用户画像特征用 Redis 还是 HBase？各自的优缺点？
- 大规模稀疏特征（如用户行为序列）怎么存？压缩方案？
- 向量特征（Embedding）用什么存储？Milvus/Faiss/自研向量引擎的对比？
- 特征存储的读写比例极高（99:1），怎么优化读性能？
- 特征缓存的一致性怎么保证？离线特征更新时如何避免缓存脏读？

### 14.3 特征一致性
- 离线训练和在线 Serving 的特征不一致（Training-Serving Skew）怎么解决？
- 特征回填（Backfill）怎么做？历史特征快照怎么管理？
- 特征版本管理？新旧特征切换时如何保证平滑过渡？

### 14.4 追问链路
- 特征分层→存储选型→读写优化→缓存一致性→Training-Serving Skew
- 实时特征→Flink 计算→Redis 写入→在线拉取→超时降级→兜底特征

## 15. AB 实验平台深度专项（推荐架构核心，来源：2024-2025 CSDN/力扣字节面经）

> AB 实验是推荐系统迭代的核心基础设施，字节内部支持数万个并行实验。

### 15.1 流量分桶与分层
- 流量分桶怎么做？用户 ID 哈希取模分桶的具体实现？
- 分层实验是什么？为什么需要层内互斥、层间正交？
- 如何验证分流的均匀性？AA 实验的作用？
- 如果某个实验需要 100% 流量怎么办？全量实验和分桶实验的切换？

### 15.2 指标评估与统计
- 实验的核心指标和护栏指标分别是什么？（CTR/时长/留存 vs 崩溃率/加载时长）
- 假设检验怎么做？t 检验和 Mann-Whitney U 检验各自适用什么场景？
- 多重比较问题怎么处理？Bonferroni 校正？FDR 控制？
- 实验样本量怎么估算？MDE（最小可检测效应）怎么定？
- 长期效应和短期效应不一致怎么办？（新奇效应/疲劳效应）

### 15.3 工程实现
- AB 实验平台的架构？SDK 端分流 vs 服务端分流的优缺点？
- 实验配置的下发和生效延迟怎么控制？
- 实验数据的采集和分析链路？实时看板怎么做？
- 如何支持数万个并行实验而不互相干扰？

### 15.4 追问链路
- 流量分桶→分层正交→AA 验证→指标评估→假设检验→多重比较校正
- 实验配置下发→SDK 分流→数据采集→实时看板→长期效应评估
