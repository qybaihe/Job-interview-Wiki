# 得物｜交易平台｜后端研发｜question-bank 候选

## 岗位方向
- 后端开发
- 服务端研发
- 交易平台 / 电商交易链路
- 社区电商 / 鉴别履约 / 风控 / 库存 / 售后

## 候选说明
- 本题库为新增公司“得物”的破冰候选包。
- 重点围绕公开可验证的业务特征抽题：**先鉴别，后发货**、潮流电商、社区内容与交易闭环、履约链路、风控、库存、售后。
- 题目尽量做到“可映射公开来源 + 可用于真实后端面试复习”。
- 当前未拿到足够可信的单场高质量真实面经，因此本轮以 `question-bank.md` 候选交付为主。

## 公开业务背景映射
- 得物是以潮流商品交易为核心的平台，公开介绍长期强调“社区 + 电商”闭环。
- 平台差异化环节是“先鉴别，后发货”，因此交易链路不只是标准下单/支付/发货，还包含**鉴别、复检、仓储、履约、逆向售后**等状态。
- 公开报道和公司介绍普遍提到其核心能力围绕：商品真实性保障、用户信任、履约效率、风控治理、售后争议处理。

## 交易平台专题
