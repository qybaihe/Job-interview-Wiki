## 岗位方向
- 后端开发
- 基础架构后端

## 高频题
- CAP、BASE、一致性协议分别适用于什么场景？
- etcd、ZooKeeper、Nacos、Redis 在注册发现/配置管理中如何选型？
- MySQL 主从延迟、双写一致性、读写分离的风险点有哪些？
- Redis 做基础组件缓存时，如何处理热点 key、大 key 和一致性问题？
- 什么是服务熔断、限流、降级、隔离？分别在哪一层做？
- MQ 在基础平台里承担什么职责？如何处理积压、丢失、重复、顺序问题？
- Java 内存模型中 volatile、synchronized、CAS 分别解决什么问题？
- JVM Full GC 频繁时，从哪几个方向排查？
- 如何设计统一日志、监控、链路追踪和告警能力？
- 如何做容量规划、灰度发布和故障演练？
- 如何做多租户隔离与资源配额？
- 如何评估一个基础服务的可用性 SLA/SLO？

- 你如何设计监控指标体系，区分症状和根因？
