# Oracle 中国 / 企业软件 / 后端开发 - Question Bank

## 基础能力
1. Oracle 这类企业软件公司和典型互联网公司后端岗位，在技术关注点上最大的差异是什么？
2. Java 服务端在企业软件场景里，为什么更强调稳定性、兼容性和可维护性？
3. JDBC 的执行流程是什么？连接池为什么能显著提升性能？
4. 讲一下事务的 ACID，以及在实际系统里最难真正保证的是哪一部分。
5. 什么是数据库执行计划？为什么同一条 SQL 在不同数据分布下效果会差很多？
6. B+ 树索引为什么适合数据库范围查询？
7. 乐观锁和悲观锁分别适合什么业务场景？
8. explain 里你最关注哪些字段来判断 SQL 是否需要优化？
